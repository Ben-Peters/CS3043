Computer and Network Security

omputer and network security in today’s day and age is more important than ever, especially when dealing with sensitive information. Malicious software can infect and compromise a computer in many different ways. In the movie Transformers(2007), there are many examples of how a system’s failure to stop an attack can result in catastrophic outcomes. While this move is fictional and the attacks demonstrated aren’t possible in real life, there are many useful parallels that can be drawn from the events that take place. The first real example of a network security breach is the attack on Qatar. While the Decepticons planned a physical attack, a near failure of the network security system almost led to the location of the AllSpark being compromised. Another example is in the pentagon scene, the defense secretary is informed of a second data breach on air force one as the Decepticons attempt to identify the AllSpark’s location once again.
